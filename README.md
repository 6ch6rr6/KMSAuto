# KMSAuto++ Portable (v1.0)

Официальный релиз удобного инструмента для настройки и управления лицензиями продуктов Microsoft. Работает без установки (Portable).

### Что входит в релиз:
* **KMSAuto++.exe** — стандартная версия программы для 32-битных (x86) систем.
* **KMSAuto++ x64.exe** — оптимизированная версия для 64-битных систем.
* **KMSAuto++.ini** — файл конфигурации (должен лежать в одной папке с запущенной программой).

---

### ⚠️ ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ (Ложные срабатывания)
Подобный софт по умолчанию классифицируется встроенным **Защитником Windows (Windows Defender)** и коммерческими антивирусами как вредоносный или потенциально опасный (HackTool). 

**Инструкция по запуску:**
1. Если браузер блокирует скачивание, разрешите сохранение файла в окне загрузок.
2. Перед распаковкой или запуском временно отключите *«Защиту в реальном времени»* в настройках вашего антивируса, либо добавьте скачанную папку в список исключений.
3. Запускайте исполняемый файл строго **от имени Администратора** (правой кнопкой мыши -> *Запуск от имени администратора*).
